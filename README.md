# Road-Accident-Severity-Prediction
Road Accident Severity Prediction using Machine Learning
Successfully completed a predictive analytics project focused on analyzing and predicting road accident severity using an Indian accident dataset. Implemented end-to-end data preprocessing, exploratory data analysis, and multiple machine learning models including Logistic Regression, KNN, Decision Tree, SVM, and Random Forest.
Random Forest achieved the best performance, highlighting the power of ensemble learning in real-world safety applications.
